# Tanzania Companion Pro — Version 31

Complete static PWA package. Version 31 restores and expands the Nikon Z8 setup-from-scratch workflow while preserving Version 30 functionality.

## Personalized camera-settings update

This packaged copy incorporates the finalized Nikon Z8 field setup: paired A–D focus modes, lens choices, metering, RAW compression, focal-length-aware Bank D shutter speeds, H.265 10-bit SDR video, built-in-microphone settings, attenuator guidance, Hi-Res Zoom, g15/g16 zebra settings, complete 12-position Photo and Video i Menu layouts, and a 512 GB CFexpress primary card with Slot 2 set to Overflow. Custom Settings names, numbers, and paths are aligned to Nikon Z8 C firmware 3.00.

The included `Tanzania_Companion_Replacement_Bank_Settings.md` is the complete structured setup document. It includes per-bank Fn2 Recall Shooting Functions values and limitations, the sub-selector-center Auto-area AF workflow, complete f2 photo and g2 video custom-control assignments, zebra color rationale, and step-by-step memory-card save/restore instructions for the completed menu setup.

AF-C priority is set to Release in Banks A–C. Focus controls use AF-ON-only back-button focus, Fn1 3D tracking, Fn2 action recall, sub-selector center Auto-area AF + AF-ON, and the 180–600mm's four L-Fn buttons as one Wide-area C1 + AF-ON group.
