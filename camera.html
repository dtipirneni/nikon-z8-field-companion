Step 0

Start from scratch safely

Before resetting: photograph any settings you may want to preserve, then use SETUP MENU → Save/load menu settings → Save menu settings. Resetting can remove bank values, button assignments and custom behavior.

Confirm firmware

SETUP MENU → Firmware version. Install Nikon's current Z8 firmware before building banks because menu labels and available options can change.

Save the existing setup

Insert a card, then SETUP MENU → Save/load menu settings → Save menu settings. Copy the resulting NCSET*** file to a computer without renaming it.

Reset Photo Shooting Menu Banks A–D

PHOTO SHOOTING MENU → Shooting menu bank. Highlight A, press O (Q), and confirm; repeat for B, C and D.

Reset Custom Settings Banks A–D

CUSTOM SETTINGS MENU → Custom settings bank. Highlight A, press O (Q), and confirm; repeat for B, C and D.

Enable extended banks

PHOTO SHOOTING MENU → Extended menu banks → On. This is required for exposure mode, shutter speed and aperture to follow the bank letters.

Build in menu order

Complete Global, Photo, Video and Custom sections below. Then configure i Menu, My Menu, banks and buttons.

Test every bank

Photograph and record a short clip in A, B, C and D. Confirm the bank letter, subject detection, image quality, shutter and lens guidance.

Protect the completed configuration

Save and restore all menu settings

The Z8 writes a small settings file to a memory card. It is not a backup of photographs or videos.

Card choice: when two cards are inserted, the settings file is saved to the card selected under PHOTO SHOOTING MENU → Primary slot selection. In this companion that is normally the 512 GB CFexpress card in Slot 1.

Save the finished setup

  1. Insert an unlocked card with free space.
  2. Confirm the Primary slot selection.
  3. Open SETUP MENU → Save/load menu settings.
  4. Select Save menu settings and confirm.
  5. Wait for completion before turning off the camera.
  6. Use a card reader to copy the NCSET*** file to a clearly named computer folder.
  7. Do not rename the NCSET*** file.

Load it back into the Z8

  1. First save the camera's current setup if it may be needed.
  2. Copy the original unrenamed NCSET*** file to the card's top level, outside DCIM.
  3. For the simplest detection, insert only that card.
  4. Open SETUP MENU → Save/load menu settings.
  5. Select Load menu settings and confirm.
  6. Wait, then turn the camera off and on.
  7. Verify every A–D bank, f1/g1 i Menu, f2/g2 controls, Fn2 recall in A–C, Fn2 Starlight view in D, and Secondary slot function Overflow.
Important limitations: the file works only with another Nikon Z8, and the camera cannot load it if its NCSET*** filename is changed. Custom Picture Controls are saved as Auto. Exact date/time, location position, timecode origin and Network Menu connection settings are not in Nikon's documented saved set. Lens-dependent VR and current exposure values should always be checked after loading.

Exposure feedback while composing

Show the live histogram while shooting

Configure the rear monitor and electronic viewfinder separately, then use DISP to show the histogram. Repeat in Custom Settings Banks A–D.

Photo — rear monitor

  1. Open d Shooting/display → d19 Custom monitor shooting display.
  2. Enable Display 2, highlight it, and press 2/right.
  3. Check Basic shooting info and Histogram.
  4. Press MENU/G to finish.
  5. While composing on the monitor, press DISP until the histogram appears.

Photo — viewfinder

  1. Open d20 Custom viewfinder shooting display.
  2. Enable Display 2 and press 2/right.
  3. Check Basic shooting info and Histogram.
  4. Put your eye to the viewfinder and press DISP until the histogram appears.

Video — both displays

  1. Open g18 Brightness information display → Histogram.
  2. Set Standard size, transparency 2 and bottom-right position.
  3. Under g20 monitor and g21 viewfinder, enable Display 2.
  4. For each Display 2, check Basic shooting info and Brightness information.
  5. Press DISP during standby or recording to show it.
How to read it: left is shadows, center is midtones and right is highlights. A pile-up touching the right edge warns of possible clipping. A left-heavy graph is normal at night; do not force Bank D toward the center. The live graph reflects the camera preview, so confirm important photos with the captured-image histogram and highlight warnings.
Display interactions: the photo-mode Video record button assigned Live view info display off hides the histogram and other overlays until pressed again. In Bank D, Adjust for ease of viewing can brighten the preview independently of exposure; Starlight view disables the live histogram entirely. Turn Starlight view Off and judge a captured test frame with playback histogram.

Bank summary

A — Safari Wildlife

Photo: High Efficiency★ RAW, AF-C, Animals, Wide C1, Matrix, 1/1600, Auto ISO, 15 fps. Video: 4K60 H.265 10-bit SDR, 1/125. Lens: 180–600mm f/5.6–6.3.

B — Birds & Action

Photo: High Efficiency★ RAW, AF-C, Birds, Wide C1, Matrix, 1/2500–1/3200, 20 fps. Video: 4K120/60 H.265 10-bit SDR. Lens: 180–600mm f/5.6–6.3.

C — Travel / People / Environment

Photo: A mode, Lossless RAW+JPEG, AF-C people or AF-S static, Matrix, Auto ISO minimum 1/500 for people. Video: 4K24/30 H.265 10-bit SDR. Lens: 70–180mm f/2.8; use a wider lens if owned, or a phone/panorama, for Stone Town interiors, groups and broad landscapes.

D — Night Sky / Low Light

Photo: Lossless RAW, MF, detection Off, f/2.8, ISO 3200–6400, 5–7 s at 70mm (start 5 s) or 1.6–2 s at 180mm, tripod VR Off. Fn2: Starlight view. Video: 4K24. Lens: 70–180mm f/2.8.

Field operation

How to switch banks correctly

Photo mode

i Menu → Photo shooting menu bank → choose letter. Then i Menu → Custom settings bank → choose the same letter.

Video mode

Move selector to video → i Menu → Video shooting menu bank → choose the matching letter.

Check before shooting

Confirm bank letter, image quality, detection type, shutter speed, Auto ISO and release mode in the display.

Extended photo menu banks

Enable only after testing. It allows exposure mode, shutter and aperture to follow the Photo bank, but can cause surprises when switching.
