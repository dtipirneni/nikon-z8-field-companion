Nikon Z8 Safari Setup PWA

To use locally:
1. Extract this ZIP.
2. Serve the folder from a local or web server. Service workers do not run from file:// URLs.
3. Open index.html in Safari/Chrome.
4. Use Add to Home Screen / Install App.

Google Drive stores these files but does not serve them as a PWA website.
For installation as a real PWA, publish the folder with GitHub Pages, Netlify, Cloudflare Pages, or similar static hosting.

The app is structured for a camera starting from factory defaults. Each Photo and Video bank includes setting values, exact menu paths, fastest controls, and local completion checkboxes.
