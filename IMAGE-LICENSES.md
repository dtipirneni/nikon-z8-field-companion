# Wildlife image licensing

Version 23.2 uses original offline SVG species icons.

Free web photographs can be added, but every image must be checked individually. For commercial use, prefer:

- CC0 / Public Domain
- CC BY, with photographer and source attribution
- CC BY-SA, only when share-alike obligations are acceptable

Do not assume every image on Wikipedia or Wikimedia Commons has the same license.

For each future photograph, record:
- local filename
- species
- photographer
- source page
- exact license
- attribution text
- date downloaded
- whether modifications were made
