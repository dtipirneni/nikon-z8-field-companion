# Tanzania Companion Pro — Version 31.1

Complete static PWA package. Version 31.1 adds a firmware 3.00 compatibility audit to the expanded Nikon Z8 setup workflow.

## Personalized camera-settings update

This packaged copy incorporates the finalized Nikon Z8 field setup: paired A–D focus modes, lens choices, metering, RAW compression, focal-length-aware Bank D shutter speeds, H.265 10-bit SDR video, built-in-microphone settings, attenuator guidance, Hi-Res Zoom constraints, g15/g16 zebra settings, complete 12-position Photo and Video i Menu layouts, and a 512 GB CFexpress primary card with Secondary slot function set to Overflow. Custom Settings names, numbers, and paths are aligned to Nikon Z8 C firmware 3.00.

The included `Tanzania_Companion_Replacement_Bank_Settings.md` is the complete structured setup document. It includes an explicit A–D f2 Photo Custom Controls matrix, per-bank Fn2 Recall Shooting Functions values, complete g2 video controls, photo/video live-histogram setup for both displays, zebra guidance, and step-by-step memory-card save/restore instructions.

AF-C priority is set to Release in Banks A–C. Focus controls use AF-ON-only back-button focus, Fn1 3D tracking, Fn2 action recall in A–C, Fn2 Starlight view in D, sub-selector center Auto-area AF + AF-ON, and the 180–600mm's four L-Fn buttons as one Wide-area C1 + AF-ON group.
