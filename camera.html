Step 0

Start from scratch safely

Before resetting: photograph any settings you may want to preserve, then use SETUP MENU → Save/load menu settings → Save menu settings. Resetting can remove bank values, button assignments and custom behavior.

Confirm firmware

SETUP MENU → Firmware version. Install Nikon's current Z8 firmware before building banks because menu labels and available options can change.

Save the existing setup

Insert a card, then SETUP MENU → Save/load menu settings → Save menu settings. Copy the resulting NCSET*** file to a computer without renaming it.

Reset photo settings

PHOTO SHOOTING MENU → Reset photo shooting menu → Yes.

Reset custom settings

CUSTOM SETTINGS MENU → Reset custom settings → Yes.

Reset video settings

Switch to video mode, then VIDEO RECORDING MENU → Reset video recording menu → Yes.

Build in menu order

Complete Global, Photo, Video and Custom sections below. Then configure i Menu, My Menu, banks and buttons.

Test every bank

Photograph and record a short clip in A, B, C and D. Confirm the bank letter, subject detection, image quality, shutter and lens guidance.

Protect the completed configuration

Save and restore all menu settings

The Z8 writes a small settings file to a memory card. It is not a backup of photographs or videos.

Card choice: when two cards are inserted, the settings file is saved to the card selected under PHOTO SHOOTING MENU → Primary slot selection. In this companion that is normally the 512 GB CFexpress card in Slot 1.

Save the finished setup

  1. Insert an unlocked card with free space.
  2. Confirm the Primary slot selection.
  3. Open SETUP MENU → Save/load menu settings.
  4. Select Save menu settings and confirm.
  5. Wait for completion before turning off the camera.
  6. Use a card reader to copy the NCSET*** file to a clearly named computer folder.
  7. Do not rename the NCSET*** file.

Load it back into the Z8

  1. First save the camera's current setup if it may be needed.
  2. Copy the original unrenamed NCSET*** file to the card's top level, outside DCIM.
  3. For the simplest detection, insert only that card.
  4. Open SETUP MENU → Save/load menu settings.
  5. Select Load menu settings and confirm.
  6. Wait, then turn the camera off and on.
  7. Verify every A–D bank, f1/g1 i Menu, f2/g2 control, Fn2 recall and Slot 2 Overflow.
Important limitations: the file works only with another Nikon Z8, and the camera cannot load it if its NCSET*** filename is changed. Custom Picture Controls are saved as Auto. Exact date/time, location position, timecode origin and Network Menu connection settings are not in Nikon's documented saved set. Lens-dependent VR and current exposure values should always be checked after loading.

Field operation

How to switch banks correctly

Photo mode

i Menu → Photo shooting menu bank → choose letter. Then i Menu → Custom settings bank → choose the same letter.

Video mode

Move selector to video → i Menu → Video shooting menu bank → choose the matching letter.

Check before shooting

Confirm bank letter, image quality, detection type, shutter speed, Auto ISO and release mode in the display.

Extended photo menu banks

Enable only after testing. It allows exposure mode, shutter and aperture to follow the Photo bank, but can cause surprises when switching.
