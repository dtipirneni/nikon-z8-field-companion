Professional workflow additions

Fast controls

These additions follow widely used Nikon Z8 field practices and the workflow themes discussed in Hudson Henry’s setup content, adapted to your safari.

Fn1 — temporary 3D Tracking

Assign AF-area mode + AF-ON → 3D-tracking. Hold Fn1 when the subject leaves your Wide C1 box or moves unpredictably.

Fn2 — Recall shooting functions

Create an emergency action override: M, 1/3200 s, wide-open aperture, Auto ISO, AF-C, Wide C1, animal/bird detection and high-speed release.

AF-ON — back-button focus

Use AF-ON for primary focusing. Consider disabling shutter-button AF only after practicing until it is automatic.

Sub-selector center

Assign subject tracking or AF-area recall so the joystick remains both a focus-point controller and an instant override.

ISO button

Use ISO + sub-command dial to toggle Auto ISO; use ISO + main dial for the base ISO when Auto ISO is off.

Extended photo menu banks

Turn this on only when you want exposure mode, shutter and aperture recalled with a Photo Shooting Menu Bank. Test carefully because bank behavior can surprise users.

Bank discipline

Name A–D consistently, copy shared settings first, then change only the settings that define each bank. Verify the Photo and Custom bank letters match before shooting.
