# Tanzania Companion Pro — Version 31

Complete static PWA package. Version 31 restores and expands the Nikon Z8 setup-from-scratch workflow while preserving Version 30 functionality.

## Camera guide
The `camera.html` guide is sorted by menu type:
1. Global / Setup
2. Photo Shooting Menu
3. Video Recording Menu
4. Custom Settings a–g
5. i Menu and My Menu
6. Bank summary and switching
7. Smart recommendations
8. Verification

Photo and video banks use matching names: A Safari Wildlife, B Birds & Action, C Travel / People / Environment, D Night Sky / Low Light.

This package uses one CSS file (`css/styles.css`) and one JavaScript file (`js/app.js`).

## Privacy
The bundled itinerary contains personal family and travel information. Do not publish this repository publicly without removing or redacting the itinerary PDF.
